# 💰 财神233 - 中国贷款与财务计算工具

一个专业的贷款计算与财务风险评估工具，帮助您更好地规划财务，合理规避风险。

## ✨ 功能特性

### 📊 核心功能

1. **商业贷款计算器**
   - 等额本息/等额本金计算
   - 详细还款计划
   - 财务风险评估
   - AI 智能建议

2. **购房贷款计算器**
   - 公积金 + 商业组合贷款
   - 首付比例计算
   - 分项贷款明细
   - 购房财务建议

3. **组合贷款计算器**
   - 灵活配置贷款比例
   - 还款方式对比
   - 利息节省分析
   - 最优方案推荐

4. **财务风险评估**
   - 综合财务健康评分
   - 收支分析
   - 债务分析
   - 应急储备评估
   - AI 深度财务规划

### 🎯 专业特性

- **财务风险管理**：基于专业的财务风险管理原则
- **负债率评估**：智能计算并评估您的负债水平
- **收入建议**：根据月供推荐合理的收入水平

## 📖 使用指南

### 1. 商业贷款计算

1. 进入"商业贷款"页面
2. 输入贷款金额、年限、利率
3. 选择还款方式（等额本息/等额本金）
4. 填写您的月收入和其他债务
5. 点击"计算贷款"查看结果
6. 查看风险评估和 AI 建议

### 2. 购房贷款计算

1. 进入"购房贷款"页面
2. 输入房屋总价和首付比例
3. 设置公积金和商业贷款额度及利率
4. 填写家庭月收入
5. 查看详细的贷款分解和风险评估

### 3. 组合贷款计算

1. 进入"组合贷款"页面
2. 选择"快速计算"或"自定义配置"
3. 调整公积金和商业贷款比例
4. 点击"对比还款方式"查看不同还款方式的差异
5. 查看利息节省分析

### 4. 财务风险评估

1. 进入"风险评估"页面
2. 详细填写收入、支出、债务、资产信息
3. 点击"开始评估"获取综合评分
4. 查看收支分析、债务分析、应急储备评估
5. 获取 AI 深度财务规划建议

## 💡 财务风险等级说明

- **低风险（负债率 ≤ 30%）**：财务状况健康，可以适度投资
- **中等风险（30% < 负债率 ≤ 50%）**：需要控制消费，避免新增债务
- **高风险（50% < 负债率 ≤ 70%）**：建议增加收入或延长贷款期限
- **严重风险（负债率 > 70%）**：必须立即调整财务计划

## 🔧 核心算法

### 等额本息计算公式

```
月供 = 本金 × 月利率 × (1 + 月利率)^还款月数 / [(1 + 月利率)^还款月数 - 1]
```

### 等额本金计算公式

```
每月本金 = 本金 / 还款月数
每月利息 = 剩余本金 × 月利率
月供 = 每月本金 + 每月利息
```

### 负债率计算

```
负债率 = (月债务总额 / 月收入) × 100%
```
